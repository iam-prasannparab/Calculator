# Arithmos Calculator

A sleek, modern calculator web application built with React, Tailwind CSS, and Framer Motion.

## Features

- **Modern UI**: Clean, minimalist design with smooth animations.
- **Dark/Light Mode**: Toggle between light and dark themes.
- **Calculation History**: Keep track of your last 10 calculations.
- **Keyboard Support**: Full keyboard support for digits and operators.
- **Responsive**: Works perfectly on mobile and desktop.
- **Precision**: Handles basic arithmetic operations with ease.

## Tech Stack

- **Framework**: [React](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [motion (Framer Motion)](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository or download the source code.
2. Open your terminal in the project directory.
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running the Application

To start the development server:
```bash
npm run dev
```
The application will be available at `http://localhost:3000`.

### Building for Production

To create a production-ready build:
```bash
npm run build
```
The output will be in the `dist` directory.
